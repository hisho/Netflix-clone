## command

### Watch

#### watch when development

```shell script
$ npm start
```

#### watch when SSG

```shell script
$ npm run serve
```

### Build

#### Build when production

### build when SSR

```shell script
$ npm run build
```

#### build when SSG

```shell script
$ npm run export
```

### Lint

```shell
$ npm run lint
```

### View Tailwindcss

```shell script
$ npm run view:tailwind
```
